### Pong game

> Simple pong game clone built with rust & tetra.

![screenshot](https://raw.githubusercontent.com/frenchbread/pong/main/resources/screenshot.png)

### clone the repo

```
❯ git clone https://github.com/frenchbread/pong.git && cd pong
```

### start

```
❯ cargo run
```
